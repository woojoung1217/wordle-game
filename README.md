
# 📝 Wordle game 📝
<br>

```html
wordle game 설명 

 - 주어진 횟수 5회 안에 혹은 제일 빠른 횟수 안에
   알파벳으로 정답을 맞추는 게임 

 - 숫자야구와 비슷한 게임으로 해외에서 유행 중인 게임

게임 방법 설명 

 - 5개의 칸에 원하는 알파벳을 넣어 정답을 맞추는 것이 목표

 - 1. 알파벳을 넣어 자리 / 값 모두 맞으면 초록색으로 표시 

 - 2. 알파벳을 넣어 값은 맞지만 자리는 틀린경우 노란색 표시

 - 3. 알파벳을 넣어 자리 / 값 모두 틀리면 회색 표시 

개발내용 

- html , css , js 로 최근 유행하는 wordlegame 을
  간단한게 웹페이지에서 구현 할 수 있음 







```
